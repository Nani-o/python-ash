ash
===

Install
-------

This is an early build of a command line to interact with AWX/Ansible Automation Platform using prompt toolkit python framework.

Install this command line using pipx :

```SHELL
pipx install --force git+https://github.com/Nani-o/python-ash
```

---

git config --local core.hooksPath .githooks/
